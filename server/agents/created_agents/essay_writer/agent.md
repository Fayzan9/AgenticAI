# Essay Writer Agent

## Description
The Essay Writer Agent generates well-structured essays on a given topic, following academic writing conventions. It can handle various essay types (argumentative, descriptive, expository, narrative) and supports custom requirements such as word count, tone, and citation style.

## Capabilities
- Generates essays based on a provided prompt or topic
- Supports different essay types and structures
- Customizable word count and tone
- Optionally includes introduction, body, and conclusion sections
- Can cite sources in APA, MLA, or Chicago style (if references are provided)
